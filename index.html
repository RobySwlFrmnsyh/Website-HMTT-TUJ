Program Kerja

Departemen Medkominfo

Program 1: Workshop Teknologi

Mengadakan workshop untuk meningkatkan keterampilan mahasiswa dalam bidang teknologi terbaru.

Program 2: Seminar Kepemimpinan

Seminar yang bertujuan untuk mengembangkan keterampilan kepemimpinan mahasiswa.

Program 3: Kompetisi Inovasi

Kompetisi untuk mendorong mahasiswa menciptakan inovasi dalam bidang teknik telekomunikasi.

Program 4: Pengabdian Masyarakat

Kegiatan sosial yang bertujuan untuk memberikan kontribusi nyata kepada masyarakat sekitar.

Departemen Hubungan Masyarakat

Program 5: Pelatihan Soft Skills

Pelatihan untuk meningkatkan kemampuan soft skills mahasiswa seperti komunikasi dan kerjasama tim.

Program 6: Diskusi Panel Industri

Diskusi dengan para ahli industri untuk memberikan wawasan tentang perkembangan terbaru dalam industri telekomunikasi.

Program 7: Kunjungan Industri

Mengunjungi perusahaan-perusahaan telekomunikasi untuk memahami proses kerja di industri.

Program 8: Kompetisi Esports

Turnamen esports untuk mengembangkan minat dan bakat mahasiswa dalam bidang game kompetitif.

Departemen Keilmuan

Program 9: Training Entrepreneurship

Pelatihan kewirausahaan untuk membekali mahasiswa dengan keterampilan dalam memulai dan mengelola bisnis.

Program 10: Lomba Karya Tulis Ilmiah

Kompetisi menulis karya ilmiah untuk meningkatkan keterampilan menulis dan penelitian mahasiswa.

Program 11: Hackathon

Hackathon untuk mengembangkan keterampilan pemrograman dan inovasi mahasiswa dalam menciptakan solusi teknologi.

Program 12: Seminar Kewirausahaan

Seminar untuk memberikan wawasan dan inspirasi kepada mahasiswa dalam bidang kewirausahaan.

Departemen Pengembangan Manusia

Program 13: Pelatihan Kepemimpinan

Pelatihan untuk mengembangkan keterampilan kepemimpinan dan manajemen mahasiswa.

Program 14: Kompetisi Olahraga

Kompetisi olahraga untuk meningkatkan kesehatan dan kebugaran mahasiswa.

Program 15: Lomba Debat

Lomba debat untuk mengasah keterampilan berpikir kritis dan berargumentasi mahasiswa.

Program 16: Workshop Pengembangan Diri

Workshop yang bertujuan untuk mengembangkan potensi diri dan kepercayaan diri mahasiswa.
